## 1. 压缩策略选择决策树

### 1.1 决策流程图

```
开始
  │
  ├─ 会话长度 < 50 消息？
  │   └─ 是 → 不需要压缩，使用完整上下文
  │
  ├─ 需要最大压缩率（99%+）？
  │   └─ 是 → 使用不透明压缩（Opaque Compression）
  │
  ├─ 文件追踪至关重要？
  │   ├─ 是 → 锚定迭代摘要 + 独立工件追踪
  │   └─ 否 → 继续判断
  │
  ├─ 会话有清晰阶段边界？
  │   ├─ 是 → 再生完整摘要（Regenerative Summary）
  │   └─ 否 → 锚定迭代摘要（Anchored Iterative）
  │
  └─ 默认推荐：锚定迭代摘要
```

### 1.2 策略对比表

| 维度 | 锚定迭代摘要 | 再生完整摘要 | 不透明压缩 |
|------|-------------|-------------|-----------|
| 压缩率 | 98.6% | 98.7% | 99.3% |
| 质量分数 | 3.70/5.0 | 3.44/5.0 | 3.35/5.0 |
| 准确性 | 4.04 | 3.74 | 3.43 |
| 上下文感知 | 4.01 | 3.56 | 3.64 |
| 工件追踪 | 2.45 | 2.33 | 2.19 |
| 可解释性 | 高 | 高 | 低 |
| 实现复杂度 | 中 | 低 | 高 |
| 适用场景 | 长会话、编码任务 | 阶段性任务 | 极限压缩需求 |

### 1.3 选择建议

**选择锚定迭代摘要，如果：**
- 会话超过 100 条消息
- 需要追踪文件修改历史
- 需要验证保留了哪些信息
- 重新获取成本高（大型代码库）

**选择再生完整摘要，如果：**
- 任务有明确的阶段划分
- 摘要可读性是首要需求
- 会话相对较短（50-150 消息）

**选择不透明压缩，如果：**
- Token 成本是主要约束
- 会话较短且重新获取成本低
- 不需要人工审查压缩内容

## 2. 锚定迭代摘要完整实现

### 2.1 核心原理

锚定迭代摘要通过**结构化章节**强制保留关键信息。每次压缩时：
1. 仅摘要新截断的内容
2. 将新摘要**增量合并**到现有章节
3. 避免完全重新生成导致的信息漂移

#### 步骤 2：Token 计数与触发检测

```python
def estimate_tokens(text: str) -> int:
    """简化的 token 估算（生产环境应使用 tiktoken）"""
    return len(text) // 4

def should_compress(self, messages: List[Message]) -> bool:
    """判断是否需要触发压缩"""
    total_tokens = sum(estimate_tokens(msg.content) for msg in messages)
    self.current_tokens = total_tokens

    # 加上现有摘要的 token 数
    summary_tokens = estimate_tokens(json.dumps(self.current_summary))
    total_with_summary = total_tokens + summary_tokens

    utilization = total_with_summary / self.context_window
    return utilization >= self.trigger_threshold
```

#### 步骤 3：增量摘要生成

```python
async def compress_messages(
    self,
    messages: List[Message],
    llm_client
) -> tuple[str, List[Message]]:
    """
    执行压缩：摘要旧消息，保留最近消息

    返回：(压缩后的摘要文本, 保留的消息列表)
    """
    # 计算截断点：保留最近 20% 的消息
    keep_count = max(10, int(len(messages) * 0.2))
    messages_to_compress = messages[:-keep_count]
    messages_to_keep = messages[-keep_count:]

    # 生成新内容的摘要
    new_summary = await self._generate_incremental_summary(
        messages_to_compress,
        llm_client
    )

    # 合并到现有摘要
    merged_summary = await self._merge_summaries(
        self.current_summary,
        new_summary,
        llm_client
    )

    # 记录压缩周期
    cycle = CompressionCycle(
        cycle_id=len(self.compression_history) + 1,
        messages_compressed=len(messages_to_compress),
        compression_timestamp=time.time(),
        summary_sections=merged_summary
    )
    self.compression_history.append(cycle)

    self.current_summary = merged_summary

    # 格式化摘要为文本
    summary_text = self._format_summary(merged_summary)

    return summary_text, messages_to_keep
```

#### 步骤 4：生成增量摘要（关键）

```python
async def _generate_incremental_summary(
    self,
    messages: List[Message],
    llm_client
) -> dict:
    """生成新消息的结构化摘要"""

    # 构建提示词
    messages_text = "\n\n".join([
        f"{msg.role}: {msg.content}" for msg in messages
    ])

    prompt = f"""Analyze the following conversation segment and extract structured information.

Conversation:
{messages_text}

Extract the following information in JSON format:
{{
  "session_intent": "What is the main goal or problem being addressed?",
  "root_cause": "Any root causes or key findings discovered?",
  "files_modified": ["List of files modified with descriptions"],
  "files_read": ["List of files read but not modified"],
  "decisions": ["Key decisions made with rationale"],
  "test_status": "Current test status",
  "blockers": ["Current blocking issues"],
  "next_steps": ["Planned next actions"],
  "technical_details": ["Important technical details: error messages, function names, etc."]
}}

Be precise with file paths, error messages, and technical terms.
"""

    response = await llm_client.generate(prompt)
    return json.loads(response)
```

#### 步骤 5：智能合并摘要

```python
async def _merge_summaries(
    self,
    existing: dict,
    new: dict,
    llm_client
) -> dict:
    """将新摘要合并到现有摘要中"""

    merged = {}

    # 对于列表类型的字段，去重合并
    list_fields = ["files_modified", "files_read", "decisions",
                   "blockers", "next_steps", "technical_details"]

    for field in list_fields:
        existing_items = existing.get(field, [])
        new_items = new.get(field, [])

        # 简单去重（生产环境应使用语义去重）
        merged[field] = list(set(existing_items + new_items))

    # 对于文本字段，使用 LLM 合并
    text_fields = ["session_intent", "root_cause", "test_status"]

    for field in text_fields:
        existing_text = existing.get(field, "")
        new_text = new.get(field, "")

        if not existing_text:
            merged[field] = new_text
        elif not new_text:
            merged[field] = existing_text
        else:
            # 使用 LLM 智能合并
            merge_prompt = f"""Merge these two descriptions into one coherent summary:

Existing: {existing_text}
New: {new_text}

Provide a merged version that preserves all important information."""

            merged[field] = await llm_client.generate(merge_prompt)

    return merged
```

```python
def _format_summary(self, summary: dict) -> str:
    """将结构化摘要格式化为 Markdown"""

    sections = []

    if summary.get("session_intent"):
        sections.append(f"## Session Intent\n{summary['session_intent']}")

    if summary.get("root_cause"):
        sections.append(f"## Root Cause / Key Findings\n{summary['root_cause']}")

    if summary.get("files_modified"):
        files = "\n".join(f"- {f}" for f in summary["files_modified"])
        sections.append(f"## Files Modified\n{files}")

    if summary.get("files_read"):
        files = "\n".join(f"- {f}" for f in summary["files_read"])
        sections.append(f"## Files Read (Not Modified)\n{files}")

    if summary.get("decisions"):
        decisions = "\n".join(f"- {d}" for d in summary["decisions"])
        sections.append(f"## Decisions Made\n{decisions}")

    if summary.get("test_status"):
        sections.append(f"## Test Status\n{summary['test_status']}")

    if summary.get("blockers"):
        blockers = "\n".join(f"- {b}" for b in summary["blockers"])
        sections.append(f"## Current Blockers\n{blockers}")

    if summary.get("next_steps"):
        steps = "\n".join(f"{i+1}. {s}" for i, s in enumerate(summary["next_steps"]))
        sections.append(f"## Next Steps\n{steps}")

    if summary.get("technical_details"):
        details = "\n".join(f"- {d}" for d in summary["technical_details"])
        sections.append(f"## Technical Details\n{details}")

    return "\n\n".join(sections)
```

## 3. 压缩触发策略与参数调优

### 3.1 触发策略对比

| 策略 | 触发条件 | 优点 | 缺点 | 推荐场景 |
|------|---------|------|------|---------|
| **固定阈值** | Token 数 > 阈值 | 简单可预测 | 可能在任务中间触发 | 通用场景 |
| **滑动窗口** | 保留最近 N 条消息 | 平滑压缩 | 需要调优窗口大小 | 长会话 |
| **基于重要性** | 压缩低相关性内容 | 保留关键信息 | 实现复杂 | 高质量需求 |
| **任务边界** | 任务完成时压缩 | 语义完整 | 时机不可预测 | 明确任务划分 |

### 3.3 参数调优指南

#### 3.3.1 触发阈值（trigger_threshold）

```python
def tune_trigger_threshold(session_logs):
    """
    根据历史会话调优触发阈值

    目标：在任务边界附近触发，避免任务中间压缩
    """
    task_boundaries = detect_task_boundaries(session_logs)
    compression_points = detect_compression_points(session_logs)

    # 计算压缩点与任务边界的对齐度
    alignment_scores = []
    for cp in compression_points:
        nearest_boundary = min(task_boundaries, key=lambda tb: abs(tb - cp))
        distance = abs(nearest_boundary - cp)
        alignment_scores.append(distance)

    avg_misalignment = np.mean(alignment_scores)

    # 如果平均错位 > 10 条消息，降低阈值
    if avg_misalignment > 10:
        return current_threshold - 0.05
    # 如果平均错位 < 3 条消息，可以提高阈值
    elif avg_misalignment < 3:
        return current_threshold + 0.05
    else:
        return current_threshold
```

## 4. 集成到 LangGraph 系统

### 4.1 LangGraph 架构概述

LangGraph 是一个用于构建有状态多智能体应用的框架。集成压缩需要：
1. 在状态图中添加压缩节点
2. 在状态中维护压缩摘要
3. 在消息传递前检查并触发压缩

### 4.2 状态定义

```python
from typing import TypedDict, Annotated, Sequence
from langgraph.graph import StateGraph, END

class AgentState(TypedDict):
```

### 4.3 压缩节点实现

```python
from langgraph.graph import StateGraph
from langchain_openai import ChatOpenAI

class CompressionNode:
    """LangGraph 压缩节点"""

    def __init__(self, config: dict):
        self.config = config
        self.llm = ChatOpenAI(model="gpt-4", temperature=0)
        self.summary_manager = AnchoredSummaryManager(
            context_window=config["context_window"],
            trigger_threshold=config["trigger_threshold"]
        )

    async def __call__(self, state: AgentState) -> AgentState:
        """执行压缩逻辑"""

        messages = state["messages"]

        # 检查是否需要压缩
        if not self.summary_manager.should_compress(messages):
            return state  # 不需要压缩，直接返回

        print(f"🗜️  Triggering compression at {len(messages)} messages")

        # 执行压缩
        summary_text, kept_messages = await self.summary_manager.compress_messages(
            messages,
            self.llm
        )

        # 更新状态
        new_state = state.copy()
        new_state["messages"] = kept_messages
        new_state["compression_summary"] = summary_text
        new_state["compression_cycle"] = state.get("compression_cycle", 0) + 1

        # 更新 token 统计
        tokens_before = sum(estimate_tokens(m.content) for m in messages)
        tokens_after = sum(estimate_tokens(m.content) for m in kept_messages)
        tokens_after += estimate_tokens(summary_text)

        new_state["total_tokens_used"] = state.get("total_tokens_used", 0) + tokens_before

        print(f"✅ Compression complete: {tokens_before} → {tokens_after} tokens "
              f"({(1 - tokens_after/tokens_before)*100:.1f}% reduction)")

        return new_state


def should_compress(state: AgentState) -> str:
    """路由函数：决定是否进入压缩节点"""

    messages = state["messages"]
    context_window = 200000
    trigger_threshold = 0.75

    total_tokens = sum(estimate_tokens(m.content) for m in messages)
    if state.get("compression_summary"):
        total_tokens += estimate_tokens(state["compression_summary"])

    utilization = total_tokens / context_window

    if utilization >= trigger_threshold:
        return "compress"
    else:
        return "continue"
```

```python
        {
            "compress": "compress",      # 需要压缩
            "continue": "tools"          # 不需要压缩，继续执行
        }
    )

    # 压缩后继续执行工具
    workflow.add_edge("compress", "tools")

    # 工具执行后返回智能体
    workflow.add_conditional_edges(
        "tools",
        should_continue,
        {
            "continue": "agent",
            "end": END
        }
    )

    return workflow.compile()


# 智能体节点示例
async def agent_node(state: AgentState) -> AgentState:
    """主智能体逻辑"""

    messages = state["messages"]

    # 如果存在压缩摘要，将其注入到系统提示中
    if state.get("compression_summary"):
        system_message = f"""You are a helpful coding assistant.

{state['compression_summary']}

"""

    else:

    # 更新状态
    new_state = state.copy()
    new_state["messages"] = [response]

    return new_state
```

### 4.5 独立工件追踪

由于工件追踪是所有压缩方法的弱项（2.2-2.5/5.0），建议在状态中独立维护：

```python
class ArtifactTracker:
    """独立的工件追踪系统"""

    def __init__(self):
        self.files_created = []
        self.files_modified = {}      # {file_path: [list of changes]}
        self.files_read = []
        self.key_entities = {}        # {entity_type: [entities]}

    def track_file_operation(self, operation: str, file_path: str, details: str = ""):
        """追踪文件操作"""
        if operation == "create":
            self.files_created.append(file_path)
        elif operation == "modify":
            if file_path not in self.files_modified:
                self.files_modified[file_path] = []
            self.files_modified[file_path].append(details)
        elif operation == "read":
            if file_path not in self.files_read:
                self.files_read.append(file_path)

    def track_entity(self, entity_type: str, entity_name: str):
        """追踪关键实体（函数名、变量名、错误消息等）"""
        if entity_type not in self.key_entities:
            self.key_entities[entity_type] = []
        if entity_name not in self.key_entities[entity_type]:
            self.key_entities[entity_type].append(entity_name)

    def get_summary(self) -> str:
        """生成工件追踪摘要"""
        lines = []

        if self.files_created:
            lines.append("### Files Created")
            lines.extend(f"- {f}" for f in self.files_created)

        if self.files_modified:
            lines.append("\n### Files Modified")
            for file_path, changes in self.files_modified.items():
                lines.append(f"- {file_path}")
                for change in changes:
                    lines.append(f"  - {change}")

        if self.files_read:
            lines.append("\n### Files Read (Not Modified)")
            lines.extend(f"- {f}" for f in self.files_read)

        if self.key_entities:
            lines.append("\n### Key Entities")
            for entity_type, entities in self.key_entities.items():
                lines.append(f"- {entity_type}: {', '.join(entities)}")

        return "\n".join(lines)
```

#### 问题 1：压缩后智能体"忘记"修改了哪些文件

#### 问题 2：压缩触发时机不当，在任务中间截断

**症状：**
```
Agent: "Let me analyze the authentication flow..."
[Compression triggered]
Agent: "What were we discussing?"
```

**原因：**
- 固定阈值触发不考虑任务边界
- 在推理过程中间压缩

**解决方案：**
```python
def should_compress_smart(state: AgentState) -> bool:
    """智能压缩触发：考虑任务边界"""

    # 基础阈值检查
    if not basic_threshold_check(state):
        return False

    # 检查是否在任务中间
    last_message = state["messages"][-1].content

    # 如果最后一条消息包含"让我"、"正在"等进行时标记，延迟压缩
    in_progress_markers = [
        "let me", "i'm analyzing", "i'm checking",
        "让我", "正在分析", "正在检查"
    ]

    if any(marker in last_message.lower() for marker in in_progress_markers):
        return False

    # 如果最后一条消息是工具调用结果，可以安全压缩
    if isinstance(state["messages"][-1], ToolMessage):
        return True

    # 如果最后一条消息包含"完成"、"done"等标记，可以压缩
    completion_markers = ["done", "complete", "finished", "完成", "好了"]
    if any(marker in last_message.lower() for marker in completion_markers):
        return True

    # 默认：达到阈值就压缩
    return True
```

#### 问题 4：压缩后 Token 数反而增加

**症状：**
```
Before compression: 150,000 tokens
After compression: 165,000 tokens (summary + kept messages)
```

**原因：**
- 摘要过于详细，超过了被压缩的内容
- 保留比例过高

**解决方案：**
```python
def compress_with_validation(messages, summary_manager):
    """压缩后验证 token 数"""

    tokens_before = sum(estimate_tokens(m.content) for m in messages)

    summary, kept_messages = summary_manager.compress_messages(messages)

    tokens_after = estimate_tokens(summary) + sum(
        estimate_tokens(m.content) for m in kept_messages
    )

    # 如果压缩后 token 数没有减少，回滚
    if tokens_after >= tokens_before * 0.95:  # 至少减少 5%
        print(f"⚠️  Compression ineffective: {tokens_before} → {tokens_after}")
        print("Skipping compression this cycle")
        return messages  # 不压缩

    print(f"✅ Compression effective: {tokens_before} → {tokens_after}")
    return [SystemMessage(content=summary)] + kept_messages
```

production_compression.py

生产级上下文压缩实现

@dataclass
class Message:

    content: str
    timestamp: float = field(default_factory=time.time)
    metadata: Dict = field(default_factory=dict)

    def token_count(self) -> int:

    cycle_id: int

    total_refetches: int = 0
    average_compression_ratio: float = 0.0
    average_quality_score: float = 0.0

    def tokens_per_task(self, total_tokens: int, tasks_completed: int) -> float:
        if tasks_completed == 0:
            return float('inf')
        return total_tokens / tasks_completed


# ============================================================================
# 工件追踪系统
# ============================================================================

class ArtifactTracker:
    """独立的工件追踪系统（不受压缩影响）"""

    def __init__(self):
        self.files_created: List[str] = []
        self.files_modified: Dict[str, List[str]] = {}
        self.files_read: List[str] = []
        self.key_entities: Dict[str, List[str]] = {}
        self.error_messages: List[str] = []

    def track_file_create(self, file_path: str):
        """追踪文件创建"""
        if file_path not in self.files_created:
            self.files_created.append(file_path)

    def track_file_modify(self, file_path: str, change_description: str):
        """追踪文件修改"""
        if file_path not in self.files_modified:
            self.files_modified[file_path] = []
        self.files_modified[file_path].append(change_description)

    def track_file_read(self, file_path: str):
        """追踪文件读取"""
        if file_path not in self.files_read and file_path not in self.files_modified:
            self.files_read.append(file_path)

    def track_entity(self, entity_type: str, entity_name: str):
        """追踪关键实体（函数名、变量名等）"""
        if entity_type not in self.key_entities:
            self.key_entities[entity_type] = []
        if entity_name not in self.key_entities[entity_type]:
            self.key_entities[entity_type].append(entity_name)

    def track_error(self, error_message: str):
        """追踪错误消息"""
        if error_message not in self.error_messages:
            self.error_messages.append(error_message)

    def get_summary(self) -> str:
        """生成工件追踪摘要"""
        lines = []

        if self.files_created:
            lines.append("### Files Created")
            for f in self.files_created:
                lines.append(f"- {f}")

        if self.files_modified:
            lines.append("\n### Files Modified")
            for file_path, changes in self.files_modified.items():
                lines.append(f"- **{file_path}**")
                for change in changes:
                    lines.append(f"  - {change}")

        if self.files_read:
            lines.append("\n### Files Read (Not Modified)")
            for f in self.files_read:
                lines.append(f"- {f}")

        if self.error_messages:
            lines.append("\n### Error Messages")
            for err in self.error_messages:
                lines.append(f"- `{err}`")

        if self.key_entities:
            lines.append("\n### Key Entities")
            for entity_type, entities in self.key_entities.items():
                lines.append(f"- **{entity_type}**: {', '.join(entities)}")

        return "\n".join(lines) if lines else "No artifacts tracked yet."

    def to_dict(self) -> Dict:
        """转换为字典"""
        return {
            "files_created": self.files_created,
            "files_modified": self.files_modified,
            "files_read": self.files_read,
            "key_entities": self.key_entities,
            "error_messages": self.error_messages
        }

# 锚定迭代摘要管理器

class AnchoredSummaryManager:
    """锚定迭代摘要管理器"""

    def __init__(
        self,
        context_window: int = 200000,
        trigger_threshold: float = 0.75,
        keep_ratio: float = 0.20,
        min_keep_messages: int = 10
    ):
        self.context_window = context_window
        self.trigger_threshold = trigger_threshold
        self.keep_ratio = keep_ratio
        self.min_keep_messages = min_keep_messages

        self.current_summary: Dict = self._init_summary_structure()
        self.compression_history: List[CompressionCycle] = []
        self.metrics = CompressionMetrics()

    def _init_summary_structure(self) -> Dict:
        """初始化摘要结构"""
        return {
            "session_intent": "",
            "root_cause": "",
            "files_modified": [],
            "files_read": [],
            "decisions": [],
            "test_status": "",
            "blockers": [],
            "next_steps": [],
            "technical_details": []
        }

    def should_compress(self, messages: List[Message]) -> bool:
        """判断是否需要压缩"""
        total_tokens = sum(msg.token_count() for msg in messages)

        # 加上现有摘要的 token 数
        if self.current_summary:
            summary_text = self._format_summary(self.current_summary)
            total_tokens += len(summary_text) // 4

        utilization = total_tokens / self.context_window
        return utilization >= self.trigger_threshold

    async def compress_messages(
        self,
        messages: List[Message],
        llm_client: AsyncOpenAI
    ) -> Tuple[str, List[Message]]:
        """
        执行压缩

        返回：(摘要文本, 保留的消息列表)
        """
        # 计算保留数量
        keep_count = max(
            self.min_keep_messages,
            int(len(messages) * self.keep_ratio)
        )

        messages_to_compress = messages[:-keep_count]
        messages_to_keep = messages[-keep_count:]

        # 记录压缩前状态
        tokens_before = sum(msg.token_count() for msg in messages)

        # 生成新内容的摘要
        new_summary = await self._generate_incremental_summary(
            messages_to_compress,
            llm_client
        )

        # 合并到现有摘要
        merged_summary = await self._merge_summaries(
            self.current_summary,
            new_summary,
            llm_client
        )

        self.current_summary = merged_summary

        # 格式化摘要
        summary_text = self._format_summary(merged_summary)

        # 记录压缩后状态
        tokens_after = (len(summary_text) // 4) + sum(
            msg.token_count() for msg in messages_to_keep
        )

        # 记录压缩周期
        cycle = CompressionCycle(
            cycle_id=len(self.compression_history) + 1,
            timestamp=time.time(),
            messages_before=len(messages),
            messages_after=len(messages_to_keep),
            tokens_before=tokens_before,
            tokens_after=tokens_after,
            compression_ratio=(1 - tokens_after / tokens_before) if tokens_before > 0 else 0,
            summary_sections=merged_summary
        )
        self.compression_history.append(cycle)

        # 更新指标
        self.metrics.total_compressions += 1
        self.metrics.total_tokens_saved += (tokens_before - tokens_after)

        return summary_text, messages_to_keep

    async def _generate_incremental_summary(
        self,
        messages: List[Message],
        llm_client: AsyncOpenAI
    ) -> Dict:
        """生成增量摘要"""

        # 构建对话文本
        conversation_text = "\n\n".join([
            f"{msg.role.upper()}: {msg.content}"
            for msg in messages
        ])

        prompt = f"""Analyze the following conversation segment and extract structured information.

Conversation:
{conversation_text}

Extract the following information in JSON format:
{{
  "session_intent": "What is the main goal or problem being addressed?",
  "root_cause": "Any root causes or key findings discovered?",
  "files_modified": ["List of files modified with brief descriptions"],
  "files_read": ["List of files read but not modified"],
  "decisions": ["Key decisions made with rationale"],
  "test_status": "Current test status (e.g., '14 passing, 2 failing')",
  "blockers": ["Current blocking issues"],
  "next_steps": ["Planned next actions"],
  "technical_details": ["Important technical details: error messages, function names, variable names, etc."]
}}

IMPORTANT:
- Be precise with file paths, error messages, and technical terms
- Include ALL files mentioned, even if briefly
- Preserve exact error messages and function names
- If a field has no information, use empty string or empty list

Output only valid JSON, no additional text.
"""

        response = await llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0
        )

        content = response.choices[0].message.content.strip()

        # 提取 JSON（处理可能的 markdown 代码块）
        if content.startswith("```"):
            content = re.sub(r'^```json?\s*', '', content)
            content = re.sub(r'\s*```$', '', content)

        return json.loads(content)

    async def _merge_summaries(
        self,
        existing: Dict,
        new: Dict,
        llm_client: AsyncOpenAI
    ) -> Dict:
        """智能合并摘要"""

        merged = {}

        # 列表字段：去重合并
        list_fields = [
            "files_modified", "files_read", "decisions",
            "blockers", "next_steps", "technical_details"
        ]

        for field in list_fields:
            existing_items = existing.get(field, [])
            new_items = new.get(field, [])

            # 简单去重（保持顺序）
            seen = set()
            merged_list = []
            for item in existing_items + new_items:
                if item and item not in seen:
                    seen.add(item)
                    merged_list.append(item)

            merged[field] = merged_list

        # 文本字段：LLM 合并
        text_fields = ["session_intent", "root_cause", "test_status"]

        for field in text_fields:
            existing_text = existing.get(field, "")
            new_text = new.get(field, "")

            if not existing_text:
                merged[field] = new_text
            elif not new_text:
                merged[field] = existing_text
            else:
                # 使用 LLM 智能合并
                merge_prompt = f"""Merge these two descriptions into one coherent summary.
Preserve ALL technical details (file paths, error codes, function names, etc.).

Existing: {existing_text}
New: {new_text}

Provide a merged version that combines both, keeping all important information.
Output only the merged text, no additional commentary.
"""

                response = await llm_client.chat.completions.create(
                    model="gpt-4",
                    messages=[{"role": "user", "content": merge_prompt}],
                    temperature=0
                )

                merged[field] = response.choices[0].message.content.strip()

        return merged

    def _format_summary(self, summary: Dict) -> str:
        """格式化摘要为 Markdown"""

        sections = []

        if summary.get("session_intent"):
            sections.append(f"## Session Intent\n{summary['session_intent']}")

        if summary.get("root_cause"):
            sections.append(f"## Root Cause / Key Findings\n{summary['root_cause']}")

        if summary.get("files_modified"):
            files = "\n".join(f"- {f}" for f in summary["files_modified"])
            sections.append(f"## Files Modified\n{files}")

        if summary.get("files_read"):
            files = "\n".join(f"- {f}" for f in summary["files_read"])
            sections.append(f"## Files Read (Not Modified)\n{files}")

        if summary.get("decisions"):
            decisions = "\n".join(f"- {d}" for d in summary["decisions"])
            sections.append(f"## Decisions Made\n{decisions}")

        if summary.get("test_status"):
            sections.append(f"## Test Status\n{summary['test_status']}")

        if summary.get("blockers"):
            blockers = "\n".join(f"- {b}" for b in summary["blockers"])
            sections.append(f"## Current Blockers\n{blockers}")

        if summary.get("next_steps"):
            steps = "\n".join(f"{i+1}. {s}" for i, s in enumerate(summary["next_steps"]))
            sections.append(f"## Next Steps\n{steps}")

        if summary.get("technical_details"):
            details = "\n".join(f"- {d}" for d in summary["technical_details"])
            sections.append(f"## Technical Details\n{details}")

        return "\n\n".join(sections)

    def export_logs(self, output_path: str):
        """导出压缩日志"""
        logs = {
            "config": {
                "context_window": self.context_window,
                "trigger_threshold": self.trigger_threshold,
                "keep_ratio": self.keep_ratio
            },
            "metrics": asdict(self.metrics),
            "compression_history": [asdict(cycle) for cycle in self.compression_history]
        }

        with open(output_path, 'w', encoding='utf-8') as f:
            json.dump(logs, f, indent=2, ensure_ascii=False)

        print(f"📝 Compression logs exported to {output_path}")

class ContextCompressionSystem:
    """完整的上下文压缩系统"""

    def __init__(self, config: Dict):
        self.config = config
        self.llm_client = AsyncOpenAI(api_key=config.get("openai_api_key"))

        self.summary_manager = AnchoredSummaryManager(
            context_window=config.get("context_window", 200000),
            trigger_threshold=config.get("trigger_threshold", 0.75),
            keep_ratio=config.get("keep_ratio", 0.20),
            min_keep_messages=config.get("min_keep_messages", 10)
        )

        self.artifact_tracker = ArtifactTracker()
        self.messages: List[Message] = []

    async def add_message(self, role: str, content: str, metadata: Dict = None):
        """添加消息到系统"""
        msg = Message(role=role, content=content, metadata=metadata or {})
        self.messages.append(msg)

        # 自动提取工件信息
        self._auto_track_artifacts(content)

        # 检查是否需要压缩
        if self.summary_manager.should_compress(self.messages):
            await self.compress()

    def _auto_track_artifacts(self, content: str):
        """自动从消息中提取工件信息"""

        # 提取文件路径
        file_pattern = r'(?:^|[\s\'"(])((?:[a-zA-Z]:)?[/\\]?(?:[\w.-]+[/\\])*[\w.-]+\.[\w]+)'
        files = re.findall(file_pattern, content)
        for file_path in files:
            if "modified" in content.lower() or "changed" in content.lower():
                self.artifact_tracker.track_file_modify(file_path, "Modified in conversation")
            elif "created" in content.lower():
                self.artifact_tracker.track_file_create(file_path)
            else:
                self.artifact_tracker.track_file_read(file_path)

        # 提取错误消息
        error_pattern = r'(?:Error|Exception|Failed):\s*(.+?)(?:\n|$)'
        errors = re.findall(error_pattern, content)
        for error in errors:
            self.artifact_tracker.track_error(error.strip())

    async def compress(self):
        """执行压缩"""
        print(f"\n🗜️  Triggering compression at {len(self.messages)} messages...")

        summary_text, kept_messages = await self.summary_manager.compress_messages(
            self.messages,
            self.llm_client
        )

        # 更新消息列表
        self.messages = kept_messages

        cycle = self.summary_manager.compression_history[-1]
        print(f"✅ Compression complete:")
        print(f"   Messages: {cycle.messages_before} → {cycle.messages_after}")
        print(f"   Tokens: {cycle.tokens_before} → {cycle.tokens_after}")
        print(f"   Ratio: {cycle.compression_ratio*100:.1f}%")

    def get_full_context(self) -> str:
        """获取完整上下文（用于发送给 LLM）"""

        parts = []

        # 压缩摘要
        if self.summary_manager.current_summary:
            summary_text = self.summary_manager._format_summary(
                self.summary_manager.current_summary
            )
            parts.append(f"# Conversation History Summary\n\n{summary_text}")

        # 工件追踪
        artifact_summary = self.artifact_tracker.get_summary()
        parts.append(f"\n# Artifact Trail\n\n{artifact_summary}")

        # 最近消息
        if self.messages:
            recent_messages = "\n\n".join([
                f"**{msg.role.upper()}**: {msg.content}"
                for msg in self.messages
            ])
            parts.append(f"\n# Recent Conversation\n\n{recent_messages}")

        return "\n\n---\n\n".join(parts)

    def get_metrics(self) -> Dict:
        """获取系统指标"""
        return {
            "total_messages": len(self.messages),
            "compression_cycles": len(self.summary_manager.compression_history),
            "total_tokens_saved": self.summary_manager.metrics.total_tokens_saved,
            "files_tracked": len(self.artifact_tracker.files_created) +
                           len(self.artifact_tracker.files_modified),
            "errors_tracked": len(self.artifact_tracker.error_messages)
        }

```python
from typing import TypedDict, Annotated, Sequence
from langgraph.graph import StateGraph, END
from langchain_core.messages import BaseMessage, HumanMessage, AIMessage, SystemMessage
import operator


# 状态定义
class AgentState(TypedDict):
    messages: Annotated[Sequence[BaseMessage], operator.add]
    compression_summary: str
    compression_cycle: int
    artifact_tracker_state: dict


# 压缩节点
class CompressionNode:
    def __init__(self, compression_system: ContextCompressionSystem):
        self.system = compression_system

    async def __call__(self, state: AgentState) -> AgentState:
        # 同步消息到压缩系统
        for msg in state["messages"]:
            if msg not in self.system.messages:
                await self.system.add_message(
                    role=msg.type,
                    content=msg.content
                )

        # 获取更新后的状态
        new_state = state.copy()
        new_state["compression_summary"] = self.system.summary_manager._format_summary(
            self.system.summary_manager.current_summary
        )
        new_state["compression_cycle"] = len(self.system.summary_manager.compression_history)
        new_state["artifact_tracker_state"] = self.system.artifact_tracker.to_dict()

        return new_state


# 构建图
def create_graph_with_compression(compression_system: ContextCompressionSystem):
    workflow = StateGraph(AgentState)

    # 添加节点
    workflow.add_node("agent", agent_node)
    workflow.add_node("compress", CompressionNode(compression_system))
    workflow.add_node("tools", tool_node)

    # 设置边
    workflow.set_entry_point("agent")

    workflow.add_conditional_edges(
        "agent",
        lambda state: "compress" if should_compress(state) else "tools"
    )

    workflow.add_edge("compress", "tools")
    workflow.add_edge("tools", "agent")

    return workflow.compile()


async def agent_node(state: AgentState):
    # 注入压缩摘要

""")

    else:
        messages = state["messages"]

    # 调用 LLM
    response = await llm.ainvoke(messages)
```
